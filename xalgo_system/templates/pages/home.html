Welcome!

This integrated system contains the backend and frontend for the latest experimental revision of the xalgorithms rule maker (xalgo-rm).

Work on this application is funded by the Xalgorithms Foundation, this application being the latest in a set of thought experiments aiming to bring about the open and publicly-visible managment of computational rules used in law, trade, and commerce.
